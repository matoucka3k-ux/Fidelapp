# FidèleApp

Programme de fidélité pour commerçants de proximité français.

## Pages

- `/` — Landing page
- `/dashboard` — Tableau de bord
- `/dashboard/encaisser` — Encaisser (QR code + scanner)
- `/dashboard/clients` — Mes clients
- `/dashboard/points` — Système de points
- `/dashboard/compte` — Mon compte (profil, abonnement, RGPD)

## Lancer en local

```bash
npm install
npm run dev
```

Ouvrez http://localhost:5173

## Déployer sur Vercel

Glissez le dossier sur vercel.com — Vite est détecté automatiquement.

## Stack

- React 18 + React Router v6
- Recharts (graphiques)
- CSS Modules + variables CSS
- Vite
- Police : Plus Jakarta Sans (Google Fonts)
